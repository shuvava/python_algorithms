# Rotate Bits

## Question

Given a number, write a function to rotate the bits (ie circular shift).

## Example

```js
rotate(‭4294901760‬, 8) = ‭4294901760‬
rotate(324508639, 12) = 3186701689
rotate(3012497280, 17) = 520185630
```
