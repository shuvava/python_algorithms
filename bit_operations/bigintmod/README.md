# Big Int Modulus

## Question

Given a list of bytes a, each representing one byte of a larger integer (ie. {0x12, 0x34, 0x56, 0x78} represents the integer 0x12345678), and an integer b, find a % b.


## Example

```python
mod({0x03, 0xED}, 10) = 5
```
